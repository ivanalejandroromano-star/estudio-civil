# EstudioPro – Ing. Civil · RO.IV.LE.

App de gestión de estudio para Ingeniería Civil. Dashboard semanal, TPs & entregas, notas por materia y recordatorios con notificaciones push.

## Cómo publicar en GitHub Pages (una sola vez)

### Paso 1 — Crear cuenta en GitHub
Entrá a [github.com](https://github.com) y creá una cuenta gratuita si no tenés.

### Paso 2 — Crear el repositorio
1. Click en **"New repository"** (botón verde arriba a la derecha)
2. Nombre: `estudio-civil`
3. Marcá **"Public"**
4. Click **"Create repository"**

### Paso 3 — Subir los archivos
En la página del repo vacío, hacé click en **"uploading an existing file"**:
- Arrastrá `index.html` y `README.md`
- Click **"Commit changes"**

### Paso 4 — Activar GitHub Pages
1. Andá a **Settings** (pestaña del repo)
2. Sidebar izquierdo → **Pages**
3. En "Branch" seleccioná **main** y carpeta **/ (root)**
4. Click **Save**

En 1-2 minutos tu app va a estar en:
```
https://TU_USUARIO.github.io/estudio-civil/
```

## Actualizar la app en el futuro
Cuando haya una versión nueva del `index.html`:
1. Entrá al repo en GitHub
2. Click en `index.html`
3. Click en el ícono del lápiz ✏️ (Edit)
4. Pegá el nuevo contenido (Ctrl+A, Ctrl+V)
5. Click **"Commit changes"**

La URL sigue siendo la misma, recargás en el celu y ya está actualizado.

## Instalación como app en el celular
- **Android (Chrome):** Abrí la URL → menú (3 puntos) → "Agregar a pantalla de inicio"
- **iPhone (Safari):** Abrí la URL → botón compartir → "Añadir a inicio"
